# 安全说明

## 支持范围

当前仓库主要维护 mac 主线版本。

如果你发现安全问题，请不要直接公开提交包含利用细节的 Issue。

## 建议的披露方式

你可以先私下联系维护者，说明：

- 问题描述
- 影响范围
- 复现步骤
- 是否涉及敏感信息泄露

在问题修复并确认发布前，建议不要公开完整利用细节。

## 敏感信息规则

请不要在以下位置提交真实敏感信息：

- 源码
- 文档
- 截图
- 日志
- 测试用例
- Release 附件

敏感信息包括但不限于：

- DeepSeek API Key
- 本地路径中的隐私信息
- 本地数据库内容
- 其他第三方平台 Token
